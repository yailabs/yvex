# Benchmark Canon

Status: Boundary foundation
Authority: CLORI repository

CLORI benchmarks measure neural execution behavior across local, localhost, LAN
and remote node postures without taking ownership of YAI policy or NET routing.

## Benchmark Scope

CLORI benchmark reports may cover:

- model artifact identity
- quantization profile
- hardware profile
- backend profile
- prompt/workload catalog entry
- prefill latency
- decode latency
- tokens per second
- memory estimates
- KV reuse behavior
- streaming overhead
- error and degraded-mode posture
- NET-compatible remote node execution

## Report Rule

Benchmark receipts are CLORI evidence. YAI may import them, but YAI remains the
authority for case use, policy interpretation, graph linkage, fact projection
and carrier decisions.
