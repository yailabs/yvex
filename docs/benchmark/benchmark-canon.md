# Benchmark Canon

Status: Boundary foundation
Authority: CLORI repository

CLORI benchmark reports must be derived from real runs. This repository does
not claim benchmark results in its bootstrap state.

YAI controls authority.
NET moves streams.
CLORI executes neural computation.

## Benchmark Scope

Future CLORI benchmark reports may cover:

- model artifact identity
- descriptor version
- quantization profile
- hardware profile
- backend profile
- prompt/workload catalog entry
- prefill latency
- decode latency
- tokens per second
- memory estimates
- KV reuse behavior
- streaming overhead
- error and degraded-mode posture
- local, localhost, LAN or remote execution posture

## Report Rule

Benchmark receipts are CLORI evidence. YAI may later import them, but YAI
remains the authority for case use, policy interpretation, graph linkage, fact
projection and carrier decisions.

No benchmark number belongs in public documentation until it is generated by a
repeatable run and linked to its workload, model, backend and hardware profile.

No inference implementation exists in this bootstrap delivery.
