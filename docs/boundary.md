# CLORI Boundary

CLORI is a standalone Neural Execution Engine, not YAI.

YAI controls authority.
NET moves streams.
CLORI executes neural computation.

## CLORI Owns

- model artifact inspection
- model descriptors
- tensor tables
- quant profiles
- architecture registry
- family profiles
- execution IR
- backend abstraction
- memory accounting
- KV cache accounting
- decoding and sampler policy
- neural output generation
- inference metrics
- benchmark receipts
- standalone CLI/server posture
- NET-compatible node adapter posture

## CLORI Does Not Own

- YAI case authority
- YAI policy
- YAI memory
- YAI graph
- YAI facts
- YAI journal
- YAI carrier authority
- YAI action approval
- NET discovery
- NET routing
- NET trust authority

No inference implementation exists in this bootstrap delivery.
